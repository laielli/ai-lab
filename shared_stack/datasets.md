## Dataset Versioning

When adding a new dataset:
1. Document the source and collection method
2. Include statistics (size, distribution, etc.)
3. Note preprocessing steps
4. Assign a version number

## Format

```markdown
### Dataset Name v1.0

- **Source**: Where it came from
- **Size**: Number of samples
- **Format**: Data format and schema
- **Location**: Path or URL
- **Notes**: Any relevant details
```
